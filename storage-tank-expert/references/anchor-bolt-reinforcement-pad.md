# 锚栓/补强圈工程经验总结

## 锚栓受力分析正确方法

储罐锚栓受力需区分**运行工况**和**水压试验工况**，两者算法相同但载荷不同：

### 核心公式
```
净上拔力 = 内压上拔力 - (罐体自重 + 液体重量)
内压上拔力 = ρ × g × H × πD²/4

若净上拔力 < 0 → 锚栓受压（构造锚栓）
若净上拔力 > 0 → 锚栓受拉（需计算截面积）
```

### 水压试验正确分析（D=6.5m, H=10m示例）
```
水重 = 3255 kN
内压上拔力 = 3255 kN （两者相等）
罐体空重 = 164 kN
净上拔力 = 3255 - (3255 + 164) = -164 kN → 受压
```

### 常见错误
将内压上拔力（3255kN）直接与罐体空重（164kN）相减，得386kN/螺栓 → 误判需要超大规格锚栓。正确做法是内压上拔力与总重量（罐+液）比较。

## NB/T 11025-2022 补强圈标准

### DN500 补强圈规格表
| 型号 | OD(mm) | ID(mm) | t(mm) | 适用压力 |
|------|--------|--------|-------|---------|
| A型 | 650 | 505 | 8 | ≤0.1 MPa（低压） |
| B型 | 700 | 505 | 10 | 0.1~0.4 MPa |
| C型 | 750 | 505 | 12 | 0.4~1.0 MPa |

### 选型原则
- 90kPa低压选A型（OD=650, ID=505, t=8）
- ID比接管外径大3~5mm（焊接间隙）
- 低压储罐补强面积远超所需（数十倍），尺寸由制造统一性决定
