# API 620 / API 650 罐底开孔（Under-bottom Connection）与排水槽

> 适用场景：储罐罐底板开孔 + 罐外集水槽（drain sump / drain pit）/ 嵌入式清扫口
> 触发关键词：罐底开孔、排水槽、集水槽、drain sump、under-bottom、sump pit、清扫口、flush-type cleanout
> 首次整理：2026-06-10

## 核心结论（一句话版）

**API 620 没有专门的 Appendix O**（under-bottom connections 专章）——真正的"罐底开孔"专章在 **API 650 Appendix O**。**行业惯例：API 620 罐底开孔设计按 API 650 Appendix O 参照**。国内配套 SH/T 3007-2014 §6.1.2 注 1 明确认可"带排水槽的排水管"做法。

## 设计流程（API 620 罐底 + 集水槽）

1. **判定开孔尺寸和位置**
   - 大开孔（≥ 30 in./750 mm）→ API 620 §5.18 延伸颈设计
   - 中小开孔 → §5.16 一般补强 + §5.19 接管颈连接
   - 位置约束：离边缘板焊缝 ≥ 12 in.，离侧壁 ≥ 600 mm（推荐）

2. **罐底补强板设计**（参照 API 650 Appendix O / Table O.1）
   - 补强板尺寸：L（长）× W²（宽），按开孔直径查表
   - 补强板厚度 T：≥ 罐底板厚度 + 腐蚀裕量
   - 与罐底焊接：**双面连续填角焊**

3. **接管颈与罐外集水槽（sump pit）**
   - 接管颈伸出 ≥ 8 in.（API 620 §1.2）
   - 罐外集水槽独立设置，槽底标高 < 罐基础垫层排水坡度（保证自流排净）
   - 集水槽材质：与罐底板同材质或高一档，加强防腐层

4. **真空盒检漏（强制）**
   - **API 620 R.B.2.5**：补强板角焊缝"无法做气压试验"，**必须**用真空盒（vacuum box）+ 渗液膜检漏
   - 这是 API 620 比 API 650 更严的地方

5. **罐底板布置限制**（API 620 §5.9.4.4）
   - 开孔位置离边缘板焊缝 ≥ 12 in.（305 mm）
   - 三板接头相互距离 ≥ 12 in.
   - 中幅板搭接 ≥ 1 in.

## 论坛工程师共识（来自 Eng-Tips / ast-forum）

> "The best design of a draw-down or draw-off sump depends on the size of tank, tank floor and foundation design. **There is no one single API-650 standard sump detail.**"

—— API 给出设计原则（Appendix O），具体细部（sump 形状/深度/盖板/连接管）由设计者按罐容/介质/基础决定。

## ⚠️ 风险提示（必须带 Sir 走查的 5 项）

1. **排水槽必须做防腐**：槽内壁做加强防腐层（环氧玻璃鳞片 / 沥青砂），否则槽内长期积水反而腐蚀罐底板
2. **可燃液体场景**：必须加防火措施（盖板/液封），连接管上**严禁装切断阀**（失火时易失效），只能装**带金属齿的快速盖板 + 铅封**
3. **真空检漏别省**：罐底开孔一旦泄漏，介质会沿罐底板下渗基础阴极保护失效，**后果难处理**
4. **小清河流域等特别排放限值**严于国标（如 COD ≤40 mg/L、NH3-N ≤2 mg/L）—— 注意地方加严条款
5. **基础沉降评估**：开孔削弱罐底与基础的连接，必须评估基础沉降对开孔接管的影响
