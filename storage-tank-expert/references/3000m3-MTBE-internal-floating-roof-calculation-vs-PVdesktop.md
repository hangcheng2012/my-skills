## 第 3 章 包边角钢计算

**L75×8，截面积 1150.30 mm²**

### 3.1 弱连接校核（GB 50341 7.1.5）
```
A ≤ mt·g / (1415·tanθ)
```
- A_actual = 1932.41 mm²（PV，含罐壁参与宽）
- A_limit = 92303.94 × 9.81 / (1415 × tan25.087°) = **1366.91 mm²** (PV: 1366.42, Δ=+0.04%)
- **不满足弱连接 → 须设紧急通气装置**

### 3.2 抗拉环（GB 50341 A.4.1）
```
Ar = (Pi - DL)·D²·10³ / (8·σ·tanθ)
```
- DL = 0.73353 kPa（罐顶板+附件固定荷载）
- Ar = (5 - 0.73353) × 17² × 10³ / (8 × 215 × tan25.087°) = **1531.25 mm²** (PV: 1526.78, Δ=+0.29%)
- A_actual = 1932.41 > 1526.78 ✓

### 3.3 微内压最大设计压力（GB 50341 A.3.2）
```
P = A·σ_y·tanθ / (200·D²) + DL
```
- A = 1932.41 mm²（实际有效截面积，**不是单角钢 1150**）
- P = 1932.41 × 345 × 0.46816 / (200 × 17²) + 0.73353 = **6.13339 kPa** (PV: 6.13347, Δ=0.00%)
- P > Pi = 5 kPa → **满足**

### 3.4 罐壁底部不被抬起（GB 50341 A.3.3）
```
Pmax = 0.00127·DLR/D² + 0.000849·DLS/D² - 0.00153·Mw/D³
```
- PV: Pmax = **2.53222 kPa**（直接采用，单位约定较细）
- Pmax < Pi = 5 kPa → 满足
- **但 Pmax < Pi 意味着须设锚固以防止底部被抬起**

### 3.5 屈曲破坏压力（GB 50341 A.3.5）
```
Pf = 1.6·Pi - 0.000746·DLR/D²
```
- PV: Pf = **7.47697 kPa**（直接采用）

### 3.6 外压作用下连接处所需截面积（GB 50341 B.2.4-2）
```
Ar = 300·PL·D·R2 / [σ]  (PL in MPa, D, R2 in m, [σ] in MPa)
```
- **R2 = Rc/sinθ = 8500/sin25.087° = 20047.17 mm** (PV)
- Ar = 300 × 0.00193353 × 17 × 20.04717 / 215.63 = **916.79 mm²** (PV: 914.64, Δ=+0.24%)
- A_actual = 1932.41 > 914.64 ✓

### 3.7 自支撑拱顶与罐壁连接处所需截面积（GB 50341 H.2.4）
```
A = 4.6·D·Rs·PL / 2.2
```
- A = 4.6 × 17 × 20 × 1.93353 / 2.2 = **1374.56 mm²** (PV: 1374.57, Δ=0.00%)
- A_actual = 1932.41 > 1374.57 ✓

### 3.8 罐壁/罐顶参与宽
```
Wc = 0.6·√(Rc·tc) = 0.6·√(8500·3.7) = 106.40 mm  (PV: 106.40) ✓
```

### 10.2 主要差异分析

1. **标称厚度选择差异**（Section 2）：PV 采用了"制造最小厚度"工程惯例（底圈≥12mm，顶圈≥6mm），而我的计算按公式需要圆整。两者都是合理的工程决策。

2. **B.2.4-2 公式中 R2 定义**（Section 3）：R2 = Rc/sinθ = 8500/sin25.087° = 20047.17 mm，**不是罐顶球面半径 20000 mm**。理解错会导致 2.3× 偏差。

3. **A.3.2 中 A 取值**（Section 3）：A 应取实际有效截面积 1932.41 mm²（含罐壁参与宽），不是单角钢 1150 mm²。

4. **A.4.1 抗拉环公式**（Section 3）：DL = 0.73353 kPa 是罐顶板+附件折算到当量液柱的固定荷载。

5. **加肋球壳折算厚度**（Section 4）：tm = 11.027 mm 是 PV 内部计算值，依赖 7 圈 × 50 条 45×8 肋的精确建模。

6. **N 值差异**（Section 6）：B.4.1 公式 N = [445·D³/(tsmin·HSE²)]^0.25，单位约定有歧义，PV 给出 N=10，按 PV 选取。

7. **中间加强圈 Arm 公式**（Section 6）：PV 实际采用 [σ]d = 215 MPa 计算 Arm，与 GB 50341 B.4.4 "0.4·σ_y" 标准文本不一致。两者都校核通过。

8. **罐底 Lm 公式**（Section 5）：PV 用 tb = 9 mm（8 mm 中幅板 + 1 mm 折算），按 8 + (12-8)/4 = 9。
