## 1. API 620 5.12.4 承压圈(压缩环) 完整计算

### 1.1 术语对照

| 用户口语 | API 620 术语 | 功能 |
|---------|-------------|------|
| **承压圈** (旧称"支承环") | Compression Ring (5.12.4) | 抵抗罐顶推力在接合处的圆周压缩力 |
| 包边角钢 | Top Angle / Compression Ring | 同上，构造形式 |
| 抗拉环 | Tension Ring | 真空工况下抵抗圆周拉力 |
| 内部柱支承 | Internal Support (附录 D) | 顶部柱支承结构（**不同概念**） |

> **关键澄清**：储罐设计中的"承压圈"几乎都是指 5.12.4 压缩环，而非内部柱支承。**注意**：旧中文文献常称"支承环"，如果用户给出罐顶有"内部柱"，那才是附录 D 的范畴，与本节 Compression Ring 无关。

### 1.2 公式集（5.12.4.1-5.12.4.3）

```
(24) 罐顶板参与宽:   w_t = 0.6 × √(R_roof × t_e)
(25) 侧壁参与宽:     w_s = 0.6 × √(R_s   × t_s)
                     t_e, t_s 为顶板/侧壁净厚 (mm - CA)
                     R_roof, R_s 为对应半径
                     单位: in (英制)
                     w_t, w_s ≥ 0.015 × R_s

(26) 圆周力:        Q = w_t × T_e + w_s × T_os − T_φ × R_s × cos(α)
                    Q > 0 → 需压缩环
                    Q < 0 → 需抗拉环

(27) 所需截面积:    A_c = Q / (S_e × E)
                    S_e 取 API 620 表 5-1 (按 Q 为拉力时)
```

### 1.3 符号约定（**最易出错**）

API 620 5.10 规定：**拉力为正、压力为负**。代入 Q 公式 (26) 时：
- 内压工况：T_e (拱顶经向) = +P×R_roof/2 (拉)，代入时取 **负号**
- 内压工况：T_φ (拱顶纬向) = +P×R_roof/2 (拉)，代入时取 **负号**
- 内压工况：T_os (侧壁环向) = +P×R (拉)，代入时取 **负号**

> 验算技巧：代入 F.4.3 例题 (P=4 psi, R_roof=600 in, R_s=375 in)，Q 应得 +264,903 lbf（**正值需压缩环**）。若你的 Q 算出负数，符号就反了。

### 1.4 单位换算表

| 量 | mm-N-MPa 制 | in-lbf-psi 制 |
|----|------------|--------------|
| 长度 | 1 mm | 0.03937 in |
| 力/长 (单位力) | 1 N/mm | 5.710 lbf/in |
| 应力 | 1 MPa | 145.04 psi |
| 面积 | 1 mm² | 1.55×10⁻³ in² |
| 力 | 1 kN | 224.81 lbf |

### 1.5 完整算例（与本会话 D=6500mm 罐一致）

**已知：**
- D = 6500 mm, R_s = 3250 mm
- H = 10000 mm
- P_design = 20 kPa (gauge)
- R_roof = 0.85D = 5525 mm
- 拱顶半夹角 α: cos α = √(R_roof² - R²)/R_roof = 0.8087
- t_roof = 6 mm, t_shell = 8 mm, CA = 1.5 mm
- Q345R, S_e = 159 MPa (= 23000 psi)
- 焊缝 E = 1.0 (100% RT)

**步骤 1：单位力 (mm-N 制)**
```
T_e  (拱顶经向) = -P × R_roof / 2 = -0.020 × 5525/2 = -55.25 N/mm
T_φ  (拱顶纬向) = -P × R_roof / 2 = -55.25 N/mm
T_os (侧壁环向) = -P × R_s     = -0.020 × 3250  = -65.00 N/mm
```

**步骤 2：单位力转 in-lbf (1 N/mm = 5.71 lbf/in)**
```
T_e  = -55.25 × 5.71 = -315.5 lbf/in
T_φ  = -55.25 × 5.71 = -315.5 lbf/in
T_os = -65.00 × 5.71 = -371.1 lbf/in
```

**步骤 3：有效板宽 (in 制)**
```
R_roof_in = 5525 / 25.4 = 217.5 in
R_s_in    = 3250 / 25.4 = 128.0 in
t_e_in    = (6 - 1.5) / 25.4 = 0.177 in
t_s_in    = (8 - 1.5) / 25.4 = 0.256 in

w_t = 0.6 × √(217.5 × 0.177) = 3.72 in = 94.6 mm
w_s = 0.6 × √(128.0 × 0.256) = 3.43 in = 87.2 mm
w_min = 0.015 × 128.0 = 1.92 in = 48.8 mm
→ w_t, w_s 均 > w_min ✓
```

**步骤 4：圆周力 Q (lbf)**
```
Q = w_t × T_e + w_s × T_os − T_φ × R_s_in × cos(α)
  = 3.72 × (−315.5) + 3.43 × (−371.1) − (−315.5) × 128.0 × 0.8087
  = −1173.6 − 1272.9 + 32659
  = +30,213 lbf ≈ 134.4 kN
```

**步骤 5：所需截面积**
```
A_c = Q / (S_e_psi × E) = 30,213 / (23,000 × 1.0) = 1.31 in² = 845 mm²
```

**步骤 6：选用角钢 (Q345R)**
| 规格 | A (mm²) | 余量 | 结论 |
|------|---------|------|------|
| L63×63×6 | 729 | −14% | 不足 |
| L70×70×6 | 816 | −3% | 不足 |
| **L75×75×6** | **880** | **+4%** | **勉强** |
| **L75×75×8** | **1150** | **+36%** | **推荐** |
| L80×80×8 | 1230 | +46% | 偏保守 |

**推荐: L75×75×8 Q345R**

### 1.6 F.4.3 验算（API 620 官方例题）

**已知**：P=4 psi 内压, D=62.5 ft, R_s=375 in, R_roof=600 in, t_top=3/16 in, t_shell=1/4 in
```
T_e = T_φ = (1/2) × 4 × 600 = 1200 lbf/in
T_os = 4 × 375 = 1500 lbf/in
cos α = 31.25/50 = 0.625

w_t = 0.6 × √(600 × 0.1875) = 6.36 in
w_s = 0.6 × √(375 × 0.25)   = 5.81 in

Q = 6.36×(−1200) + 5.81×(−1500) − (−1200) × 375 × 0.625
  = −7632 − 8715 + 281,250
  = +264,903 lbf ✓ (正值, 需压缩环)

A_c = 264,903 / 23,000 = 11.52 in² = 7432 mm²
→ 需大型角钢或组合截面
```

### 2.2 设计压力选取流程

```
P_design = max(以下各工况):
  ① 呼吸阀定压 × 1.1 (或 + 5 kPa, 取大)
  ② 工艺操作压力 + 5 kPa (含液柱静压)
  ③ N₂ 密封压力 + 5 kPa
  ④ 泵出口压力 (连通储罐时)
  ⑤ 试验压力: 1.25 × P_design
```

**典型己烷罐设计压力：** 20~30 kPa（呼吸阀 + N₂ 封 + 10% 裕量）
- RVP 17 kPa, 呼吸阀 +20 kPa 定压, N₂ 封 2~3 kPa → P_design = 20 kPa
- 临界 RVP 介质（苯 22 kPa, 丙酮 70 kPa）→ P_design = 30~50 kPa

### 2.3 设计外压 P_vac 选取

```
P_vac = max(以下各工况):
  ① 呼吸阀负压定压 (通常 −0.5 kPa)
  ② 出料瞬态: 与泵特性曲线耦合
  ③ 火灾工况: 油气受热膨胀（通常不设真空）
  
典型取值: P_vac = 0.5 kPa (通风储罐)
         P_vac = 0.2 kPa (N₂ 封储罐)
```

## 3. 真空稳定性校核完整流程（API 620 + GB 50341）

### 3.1 公式选择决策表

| L/D | 适用公式 | 备注 |
|-----|---------|------|
| L/D ≥ 5 | Bresse: P_cr = 2.6 E (t/D)³ | 长圆筒 |
| 1 ≤ L/D < 5 | **Von Mises: P_cr = 2.6 E (t/D)³ × [(D/L)² + 2(L/D)] / (1−ν²)** | 短圆筒（储罐常见） |
| L/D < 1 | 端板屈曲控制 (D/t 敏感) | 极少储罐 |
| 任意 | **GB 150.3 外压图表法** | 最通用，需图表 |

### 3.2 储罐设计典型 L/D 范围

| 罐型 | 典型 L/D | 推荐公式 |
|------|---------|---------|
| 立式拱顶 (D=6.5, H=10) | 1.54 | Von Mises |
| 立式拱顶 (D=15, H=15) | 1.0 | Von Mises |
| 立式拱顶 (D=30, H=20) | 0.67 | 端板控制 |
| 卧式 | 3~5 | Von Mises |

### 3.3 安全系数要求

| 标准 | 安全系数 | 备注 |
|------|---------|------|
| GB 50341 | SF ≥ 3.0 | 真空稳定性 |
| API 650 附录 | SF ≥ 1.5 | 真空（更宽松） |
| 工程经验 | SF ≥ 4.0 | 推荐保守值 |

### 3.4 不满足时的处置（优先级）

1. **调整 P_vac**（减小设计真空）— 最经济
2. **增大壁厚** — P_cr ∝ t³, 效果立竿见影
3. **加设中间加强圈** — GB 50341 7.2.5
4. **改用正压呼吸阀** — 工程不推荐
5. **提高基础约束刚度** — 间接效果

## 8. 90 kPa 接近 API 620 上限时的工程变更要点

本节来自 2026-06-03 同日第二次会话: 用户将设计压力从 20 kPa 升至 **90 kPa (接近 API 620 上限 100 kPa)**, 同时指定罐底 10mm Q345R / 罐顶 8mm / 罐壁底圈 10mm。此工况触发了一系列原 20 kPa 设计未涵盖的关键工程问题, 应作为后续 90 kPa 等级 API 620 设计的标准检查表。

### 8.1 罐壁厚度: 仍由最小板厚控制, 不必加厚

| 部位 | 底圈最大压力 | t_calc | t_min (R<7.62m) | 选用 |
|------|-------------|--------|----------------|------|
| 底圈 | 154.75 kPa | 4.66 mm | 7.9 mm | **8 mm** (不变) |
| 顶圈 | 90.00 kPa | 3.34 mm | 7.9 mm | **8 mm** (不变) |

底圈环向应力 σ_θ = 77.4 MPa, 利用率仅 **48.7%**, 90 kPa 仍有较大裕量。
> **工程含义**: 中小直径 (D<7.62m) 储罐在 API 620 压力范围内, 罐壁壁厚基本不由强度控制, 而由最小板厚 (5.10.4 + 表5-6) 控制。90 kPa 等级不必将壁厚提升至 10mm。

### 8.2 浮力 > 自重: 锚栓从构造化变为承载化 ⚠ 关键

**这是 90 kPa 等级最关键的工程变化。**

| 工况 | 抗浮力 W (kN) | 浮升力 F_buoy (kN) | 净浮力 F_net | 结论 |
|------|-------------|------------------|--------------|------|
| 20 kPa 运行 | 2405 | 684 | −1721 | 构造配置 |
| **90 kPa 运行** | 2441 | 3007 | **+566** | **锚栓必须承载** |
| 90 kPa 水压试验 (112.5 kPa) | 3557 | 3753 | +196 | 锚栓承载 |

**判定流程**:
```
对每个工况计算: F_net = P × A_cross + 1.25 × F_wind − (W_tank + W_liquid)
若 F_net > 0 → 锚栓必须按承载型设计, 详细计算 F_bolt = F_net / n
若 F_net < 0 → 锚栓按 API 620 5.11.4 构造配置 (≥ 8 个 M24)
```

**90 kPa 等级锚栓计算 (D=6.5m, 16 根 M30 Q345B):**
- 单根许用承载力: F_cap = 0.9 × S_e × A_net = 0.9 × 159 × 564 = 80.7 kN
- 单根实际拉力: F_bolt = 566 / 16 = 35.4 kN
- 单根安全系数: SF = 80.7 / 35.4 = 2.28 ≥ 2.0 ✓
- 水压试验时单根拉力: 196 / 16 = 12.3 kN, SF = 6.57 (大幅富裕)

> **关键提示**: 90 kPa 等级下, **运行工况 (而非水压试验) 控制锚栓设计**。水压试验时浮力被水重补偿, 反而更安全。这是反直觉的, 详设阶段须特别说明。

### 8.3 压缩环截面积 4.5 倍增加: 需用大截面/组合件

| 设计压力 | T_e (N/mm 拉) | Q (kN/m) | A_c (mm²) | 推荐截面 |
|---------|---------------|----------|-----------|----------|
| 20 kPa | 55.3 | 134 | 848 | L75×75×8 (1150) |
| **90 kPa** | **248.6** | **604** | **3781** | **16mm 圈板 × 240mm 高 (3840) 或 25a# 槽钢 (3990)** |

**90 kPa 等级下单根角钢已不经济**, 须用以下形式之一:
- **方案A (大角钢)**: L150×150×14, A=4057 mm²
- **方案B (槽钢)**: 25a# 槽钢, A=3990 mm²
- **方案C (圈板, 推荐)**: 罐壁顶圈外侧加焊 16mm 厚 × 240mm 高圈板, 上下双面连续角焊
- **方案D (组合)**: 角钢 + 板, T 形截面

**方案C优势**: 制造灵活, 不依赖型钢库存, 适合中大型化工厂项目。

### 8.4 无边缘板 + 90 kPa 浮力: 必须加抗剪板

**用户口语 "无边缘板" + "有锚固" 即 API 620 5.11.3 条款。** 90 kPa 等级下, 由于浮力 > 自重, 抗剪板成为浮力传递关键节点, 必须详细核算。

**抗剪板设计 (16 组, 每组对应 1 根锚栓):**
```
单根锚栓拉力: F_bolt = 35.4 kN
抗剪板规格: 厚 7mm, 高 200mm, 罐壁底圈外侧
焊缝: 双面角焊, K = 7mm, L_w = 2 × 200 = 400 mm
焊缝剪应力: τ = F_bolt / (0.7 × K × L_w)
          = 35400 / (0.7 × 7 × 400)
          = 18.1 MPa
许用剪应力: 0.6 × S_e = 0.6 × 159 = 95.4 MPa
校核: 18.1 < 95.4 MPa ✓
```

**检测要求**:
- 抗剪板与罐壁底圈: 100% MT + 20% UT
- 抗剪板与罐底: 100% MT + 20% UT (焊后真空试漏)
- 抗剪焊缝是浮力工况薄弱环节, 详设必须出节点详图

### 8.5 罐壁/罐底厚度差处理 (3 种方案)

90 kPa 罐 (罐底 10mm, 罐壁 8mm) 时, 厚度差 Δt = 2mm = 25% 较薄板厚, **API 620 6.16.2 处于临界点**, 必须选择一种处理方式:

| 方案 | 内容 | 焊接 | 工艺 | 成本 | 推荐度 |
|------|------|------|------|------|--------|
| **A** 罐底削薄 | 罐底边缘 100mm 范围外侧削薄至 8mm, 1:50 缓坡 | 需现场机加工 | 削薄后 100% MT | +0.2 万 | ★★★ |
| **B** 罐壁底圈加厚 ★ | 罐壁底圈 (1429mm 高) 8→10mm, 与罐底等厚 | K形坡口全焊透, 无削薄 | 8mm 与 10mm 过渡段 75mm 缓坡 | +458 kg × 6元 = +0.28 万 | ★★★★★ |
| C 双加厚 | 罐壁底圈+罐底都升 12mm | 过于保守 | — | +1.5 万 | ★★ |

**推荐方案B (罐壁底圈局部加厚)**:
- 罐壁底圈 8mm → 10mm (Q345R), 高度 1429mm
- 与上层 8mm 罐壁对接: 削薄过渡 ≥ 75mm (1:50 缓坡, 100% MT)
- 与罐底 10mm 等厚: K 形坡口全焊透, 100% RT + 100% UT
- 抗剪板位于底圈外侧, 厚 7mm, 高 200mm

**罐体自重变化**:
- 罐壁底圈 8→10mm 增加 458 kg (罐体总重 +2.6%, 对浮力/锚栓影响微弱)
- 工程意义: 简化焊接工艺 + 增强底圈腐蚀裕量 (CA 1.5mm 在底圈升至 2.5mm 等效裕量)

### 8.6 罐顶厚度对压缩环的微弱影响

| t_roof (mm) | t_e (mm 净) | w_t (mm) | Q (kN/m) | A_c (mm²) |
|-------------|------------|----------|----------|-----------|
| 6 | 4.5 | 94.6 | 604.4 | 3811 |
| **8** | 6.5 | 113.7 | 599.6 | 3781 (-0.8%) |

**结论**: 罐顶从 6mm 加到 8mm, 压缩环 A_c 仅减小 0.8%, **不需更换原方案**。罐顶加厚主要考虑: ①刚度; ②腐蚀裕量; ③工艺统一。

### 8.7 整体 Q345R 统一材质

**用户场景**: 罐壁 Q345R + 罐底 10mm Q345R + 罐顶 8mm Q345R, 全部统一 Q345R。

**优势**:
- 焊材单一 (E5015/E5016), 焊接工艺评定简化
- 采购单一材质, 减少备料
- 罐底/罐顶/罐壁许用应力统一 (S_e = 159 MPa)
- 内部检验简化 (材料标记、复验)

**成本**: 罐顶/罐底 Q235B → Q345R 单价高 30~50%, 但总量小 (3~5 t), 总增量 < 1 万元。

### 8.9 90 kPa 等级关键检查清单 (供详设)

- [ ] 设计压力校核: 工艺包最大操作压力 + 5 kPa
- [ ] N₂ 封压力: 与 P_design 匹配 (略低 1~2 kPa)
- [ ] 呼吸阀口径: 按 API 2000 重算 (90 kPa 进出气量大幅增加)
- [ ] 罐壁厚度: 通常 8mm 即可, 详设时复核 t_calc
- [ ] 罐底厚度: 10mm Q345R 抗弯 + 抗剪储备充足
- [ ] 罐顶厚度: 8mm Q345R 应力利用率 24%, 富裕
- [ ] 压缩环: 16mm 圈板或 25a# 槽钢 (避免单角钢)
- [ ] 锚栓: 16 根 M30 起, 单根力按运行工况算
- [ ] 抗剪板: 7mm×200mm×16 组 (与锚栓对应), 焊缝 100% MT
- [ ] 罐底/罐壁厚度差: 优先方案B (底圈加厚)
- [ ] 大开孔 (DN500 人孔) 补强圈: 厚度 10→12mm
- [ ] 试压方案: 112.5 kPa, 试压期间锚栓承载, 须先装锚栓后试压
- [ ] 浮力 > 自重: 基础设计须考虑上拔, 锚栓锚固长度+握裹力核算
