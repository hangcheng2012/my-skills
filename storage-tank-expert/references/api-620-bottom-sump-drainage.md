# API 620 罐底开孔 / 排水槽 / Sump 设计参考

> **适用范围**: API Std 620 大型焊接低压储罐 (≤100 kPa 内压), 用户提出"罐底开孔接排水槽 / 集水槽 / 排水井"需求时的**合规审查 + 设计要点清单**.
>
> **KB 状态**: 本地知识库 `API STD 620-2013 (中文 OCR 版)` 中**无 sump / drain pit 独立专章**, 但相关条款齐. 本文件 = 条款梳理 + 国内规范 + 工程经验.

---

## 1. 适用条款 (API STD 620-2013)

### 1.1 罐底板定性 (5.9.4.1) — 关键基线

> "采用环墙式基础、级配基础或混凝土板基础均匀支承的圆筒形储罐的平底, 由于是利用基础支承的, **被视为非应力性隔板**."

**工程含义**: 罐底板上的开孔不会削弱承压壳体的整体强度. 但开孔处的应力集中仍需按 5.16/5.18 校核, 且不可豁免检漏.

### 1.2 开孔补强条款路由

| 开孔尺寸 | 适用条款 | 关键要求 |
|---------|---------|---------|
| **小开孔** (≤ 罐壁补强阈值) | 5.16 (单/多开孔补强) | 补强面积按 5.16.5.1: `A_r = (d₀ + 2ot - tE) × c` |
| **罐底中心大开孔** | 5.18 (罐顶/罐底中心大开孔) | 转角半径 ≥ 6% 开孔直径; 颈部按 5.10 算厚度 |
| **接管颈通用** | 5.19 (接管颈及与储罐的连接) | 接管伸出 ≥ 8 in. (外); 法兰 ASME B16.5; 双面填角焊 |

### 1.3 罐底最小厚度 (5.9.4.2)

> "所有底板均应具有至少 **1/4 in.** 的标称厚度, 不包括腐蚀裕量."

例外 (Q.3.4.7): 镍/不锈钢 3/16 in. 可.

### 1.4 焊缝检验 (R.B.2.5 / Annex R) — 罐底开孔的硬约束

> "不允许在补强板背后加气压的罐底开孔角焊缝, **必须采用真空盒检漏 (vacuum box inspection) + 渗液试验**."

**根因**: 罐底板下面没有气密空间, 无法用气压试验; 一旦泄漏, 介质沿罐底板下渗 → 基础阴极保护失效, 后果难处理. **不可省**.

### 1.5 接管伸出限制 (§1.2)

接管 > 2 in. 时, 须延伸最少 8 in. 至罐壁外, 配螺栓法兰.

## 2. 国内规范配套 (石化项目, 必引)

### 2.1 主标 (石化项目, Sir 铁律)

**SH/T 3046-2024**《石油化工立式圆筒形钢制焊接储罐设计规范》— 罐底设计/开孔必须符合此标 (石化项目, Sir 2026-06-04 明确为主标).

### 2.2 并行标

**GB 50341-2014**《立式圆筒形钢制焊接油罐设计规范》— 与 SH/T 3046 双标准执行.

### 2.3 储运系统配套标 (Sir 关键引用)

**SH/T 3007-2014**《石油化工储运系统罐区设计规范》§ 6.1.2 注 1:
> "原油、重油和易聚合的液体储罐宜设置清扫孔, 轻质油品储罐宜设置排污孔. **轻质油品储罐设有带排水槽的排水管时, 可不设置排污孔.**"

**工程含义**: 国内规范明确认可"带排水槽的排水管"是允许的工程做法 (轻质油品罐可省排污孔).

### 2.4 罐区设计补充

- **GB 50074-2014**《石油库设计规范》
- **GB 50160-2018**《石油化工企业设计防火规范》
- **AQ 3063-2025**《化工企业可燃液体常压储罐区安全管理规范》§ f: 罐壁开孔焊缝及补强圈角焊缝在第一层焊完后必须外观检查合格方可继续施焊

## 3. 设计要点清单 (踩坑防御)

### 3.1 开孔位置

- 离边缘板 ≥ 600 mm (5.9.4.4 限制三板接头距离)
- 离侧壁 ≥ 600 mm
- 避开中幅板焊缝搭接区

### 3.2 排水槽材质与防腐

- 排水槽本体材质: 与罐底板同材质或高一档 (Q235B / Q345R)
- 槽内壁防腐层: 环氧玻璃鳞片 / 沥青砂 加强 (≥ 罐底板防腐等级)
- **核心风险**: 槽内长期积水若防腐不到位, 反而腐蚀罐底板 → 罐底板下阴极保护失效
- 排水槽底部宜设放空/排净结构, 避免长期积水

### 3.3 排水槽与罐基础关系

- 槽底标高宜低于罐基础垫层排水坡度 (自流排净)
- 槽外壁距罐基础混凝土环墙 ≥ 200 mm
- 槽区做防渗层 (防介质渗入基础)

### 3.4 可燃液体场景特殊要求 ⚠

- **连接管上严禁装切断阀** (失火时易失效) — 只能用带金属齿的快速盖板 + 铅封
- 排水管必须**带排水槽** (SH/T 3007 §6.1.2 注 1 要求)
- 槽周边严禁积聚可燃蒸气 — 槽加盖/液封/通风
- 排水操作纳入工艺操作规程, 明确"先脱水后采样"程序

### 3.5 焊接与检验

- 罐底开孔补强板**角焊缝必须 100% MT + 100% 真空盒检漏** (R.B.2.5)
- 罐底开孔接管颈**对接焊缝 100% RT + 100% UT**
- 罐底开孔**禁止背后加气压试验** (无气密空间)

## 4. 设计决策流程 (Sir 工作流)

```
Q1: 储罐是 API 620 范围 (P ≤ 100 kPa, 大型焊接立式)?
   ├─ 否 → 走 GB 50341 / SH/T 3046 / SH/T 3167 / SY/T 0608 路径
   └─ 是 ↓

Q2: 介质是否易燃易爆 / 可燃液体?
   ├─ 是 → 必须 3.4 节全部可燃液体特殊要求 (盖板/液封/严禁阀/防火)
   └─ 否 → 普通工业介质

Q3: 开孔尺寸?
   ├─ 小开孔 (DN ≤ 200, 或按 5.16 阈值) → 按 5.16 单/多开孔补强
   ├─ 中开孔 (中心/罐底附近) → 按 5.18 大开孔
   └─ 大开孔 (≥ 中心大开孔阈值) → 5.18 完整 + 5.10 颈部计算

Q4: 罐底板是否承压?
   └─ 否 (5.9.4.1 明确非应力隔板), 但开孔处仍按 5.16/5.18 校核

Q5: 检验方式?
   └─ 真空盒检漏 + 渗液 (R.B.2.5, 不可省)

Q6: 排水槽结构?
   ├─ 材质: 与罐底同 (Q235B/Q345R)
   ├─ 防腐: 环氧玻璃鳞片 + 槽底排净
   └─ 与基础关系: 自流坡度, 防渗层, 距混凝土环墙 ≥ 200mm
```

## 5. 与常压储罐 (API 650) 对比

| 项 | API 650 / GB 50341 (常压 ≤18 kPa) | API 620 / SH/T 3167 (低压 18~100 kPa) |
|----|-----------------------------------|--------------------------------------|
| 罐底板定性 | 非应力隔板 (基础支承) | 非应力隔板 (基础支承) |
| 罐底最小厚 | 6 mm (碳钢) | 1/4 in. (6.35 mm) |
| 排水管/排污孔 | SH/T 3007 §6.1.2 表规定 | 同 (但 18~100 kPa 罐多为轻质油品, 可不设排污孔) |
| 开孔补强 | API 650 3.7 / GB 50341 §7.1.5 | API 620 5.16/5.18 / SH/T 3167 §6.x |
| 真空检漏 | API 650 真空试验 | R.B.2.5 (罐底开孔必做) |
| 防火间距 | GB 50160-2018 按介质分类 | 同 |

**核心结论**: API 620 与 API 650 在罐底开孔上**要求基本一致** (均视为非应力隔板 + 真空检漏), 差异主要在补强计算公式.
