## 三、B.2 单元力的计算

### 3.1 1-1 平面压力

```
p_hydro = ρ·g·H_a = 659.6 × 9.81 × 9.45 / 10⁶ = 0.0611 MPa
p_total = p_a + p_hydro = 0.09 + 0.0611 = 0.1511 MPa
```

### 3.2 罐壁横截面积

```
Ae = π × R² = π × 3250² = 33.183 × 10⁶ mm²
```

### 3.3 单元力 (SH/T 3167 B.2-1, B.2-4)

```
T1 = (R/2) × [p + (W+F)/Ae]   (轴向, W+F 向下取负)
   = 1625 × [0.1511 − 2 286 700 / 33.183×10⁶]
   = 1625 × [0.1511 − 0.0689]
   = 1625 × 0.0822
   = 133.63 N/mm²                                    ...(B.2-3)

T2 = p·R = 0.1511 × 3250 = 491.23 N/mm²             ...(B.2-4)
```

**B.2 结论**：
- T1 = 133.63 N/mm² (轴向, 压力控制)
- T2 = 491.23 N/mm² (环向, 压力控制)
- **与 PVdesktop T1=135.40/T2=491.16 差异 < 1.4%**（W 略差）
- 罐壁厚度由 T2 控制 ✓

### 6.1 输入参数 (PVdesktop 给出)

| 弯矩 | 数值 (N·m) | 说明 |
|------|-----------|------|

| M_F (起抗倾覆的介质) | 286 988.45 | 有效介质重 |

### 6.2 校核式 1

```
0.6·M_w + M_Pi  ≥  M_DL/1.5 + M_DLR
10 037 225.93  ≥  526 961.97   → 不等式成立
```

**结论：罐不稳定，需设锚固结构**

### 6.3 校核式 2

```
M_w + F_p·M_Pi  ≥  (M_DL + M_F)/2 + M_DLR
4 434 381.55  ≥  562 002.85   → 不等式成立 (F_p = 0.4)
```

**结论：需采取以下措施之一**：
- (a) 修改边缘板，使其满足该条件
- (b) 设置锚固结构

### 6.4 边缘板宽需求 (B.6 二阶措施)

```
L = 0.0291·t_a·(F_y/H)^0.5
  = 0.0291 × 10 × √(345/10.21)
  = 0.0291 × 10 × 5.815
  = 1.692 m
```
