> **方向约定**（Sir 2026-06-04 校对）：
> - 自由体分析中，(W+F) 方向**向下**（重力），代入公式时取**负值**
> - 公式中的 `+ (W+F)/Ae` 等效于 `p − |W+F|/Ae`（代数效果相同）
> - 内压 p 方向**向上**，取正值
> - 算例：W+F = **−2.11×10⁷ N**（液体 + 罐体总重），p = +0.117 MPa

## 5. B.2 单元力的计算（P41–P42）

### 5.1 标准公式

按式（5.3.2-1）和式（5.3.2-2）：

```
T1 = (R/2) × [p + (W+F)/Ae]                      ...(B.2-1)
T2 = p·R                                          ...(B.2-2)
```

> **注**：原标准 5.3.2-1 在自由体 1-1 平面，p 为该平面上的设计压力（静液柱 + 设计内压），(W+F) 为重力方向作用于自由体上的力，**取负值代入**（详见第 2 节方向约定）。

### 5.2 符号说明

| 符号 | 含义 | 本算例取值 |
|------|------|-----------|
| R | 所考虑水平面上罐壁中线半径 | R = 8 200 mm |
| p | 1-1 平面上静液柱压力 | 1100×9.81×9×10⁻⁶ = **0.097 MPa** |
| pa1 | 1-1 平面上的附加压力（按 0.9 充装系数，最高液位 8 100mm） | 1100×9.81×0.097×10⁻⁶ = **0.097 MPa**（≈ p） |
| pa | 罐壁设计温度下的罐壁设计内压力 | **0.02 MPa** |
| p' | p + pa | **0.117 MPa** |
| W | 罐内液体重量 + 罐体和罐顶重量 = **2.11×10⁷ N**（v4 Sir 校对：液体+罐体总重） | W = 2.11×10⁷ N |
| F | 罐体内部和外部材料、支架、底板等作用在自由体上的垂直分力总和（不含壳体本身） | 按式（5.3.2-3）取 |
| Ae | 1-1 水平面上罐壁内部的横截面积 = π·R² | Ae = π × 8200² ≈ **2.11×10⁸ mm²** |

### 5.3 单元力数值（P42 末）

```
T1 = (R/2) × [p + (W+F)/Ae]
   = 4100 × [0.117 − 2.11×10⁷ / 2.11×10⁸]
   = 4100 × [0.117 − 0.100]
   = 4100 × 0.017
   = **70.37 N/mm²**                              ...(B.2-3)

T2 = p·R = 0.117 × 8200 = **960.376 N/mm²**        ...(B.2-4)
```

> **关键点**：T2 = 960 N/mm² >> T1 = 70 N/mm²，原因是 T1 是 (W+F)/Ae 与 p 共同作用的纵向净力（液体+罐体重力大幅抵消了 p 的效果），T2 是 p·R 的纯周向薄膜力。
> **符号约定**：罐壁轴向压力计算时取 T1，罐壁环向薄膜应力取 T2。

## 7. B.4 抗风圈/承压圈校核（P43–P45）

### 7.1 几何公式（图 B.4-1 承压圈结构示意）

```
Wh = 0.6 × √(R1·(tq − c))                        ...(B.4-1)  [罐顶参与宽]
Wc = 0.6 × √(R2·(td − c))                        ...(B.4-2)  [罐壁参与宽]
```

**Wh / Wc 映射**（v4 关键发现）：

| 公式 | 含义 | 几何位置 | 半径 | 厚度 | 对应面积 |
|------|------|---------|------|------|---------|
| **Wh** | 罐顶参与宽 | 承压板向上 150 mm | R1=13 120 (罐顶) | tq=顶板 | **At**=(Wh+150)·tq |
| **Wc** | 罐壁参与宽 | 承压板向下 50 mm | R2=8 200 (罐壁) | td=顶圈壁 | **Ac**=Wc·td |

**参数取值**：
- R1 — 顶圈与罐壁连接处罐壁中线半径的圆周方向：**R1 = 13 120 mm**
- R2 — 罐壁中线半径：**R2 = 8 200 mm**
- tq — 顶板厚度
- td — 顶圈罐壁厚度
- c — 厚度附加值，c = 0
- **Wh = 194.38 mm**（罐顶参与宽）
- **Wc = 133.086 mm**（罐壁参与宽）

### 7.2 单元力计算公式

按式（5.4.3.1-1）、（5.4.3.1-2）、（5.5.2.3-1）：

```
T1 = (R/2) × [p + (W+F)/Ae]                       ...(B.4-3)  [式 (5.4.3.1-1)]
T2 = R·p − T1                                     ...(B.4-4)  [式 (5.4.3.1-2)]
T3 = R·pa                                         ...(B.4-5)  [式 (5.5.2.3-1)]
Q  = T1·Wh + T2·Wc − T3·R·sin α                   ...(B.4-6)
```

> **⚠️ v6 关键公式修正（2026-06-04 Sir 用 6.5m 己烷罐校核发现）**：
> - 公式 B.4-6 末项是 **`T3·R·sin α`**（R 为罐壁中线半径）
> - **不是** `T3·Ar·sin α`（Ar 为拱顶投影面积）
> - 本算例 P44 OCR 写 "T3·Ar·sin α" 是 **OCR 误读**（R 与 Ar 字符相近）
> - 标准 SH/T 3167 5.5.2.3-1 原文（OCR line 1022）确认：`O=ETW+T2sW-T.Rsing` = `Q = T1·W1 + T2·W2 − T3·R·sinα`
> - **影响**：用 R 公式算 Q 比用 Ar 公式算大 ~10×（Ar 多 1 个 R 因子）
> - 算例 Q 数值 0.802 MN (P44) 用 R 公式重算约 0.73 MN，差 ~10%，可能是算例 OCR 中 sin α 或 R 数值有微差

> **注意**：式 B.4-6 中 **Wh、Wc 为几何尺寸**（罐顶/罐壁参与宽），**与标准 5.3.2 中 W1、W2（重量：液体/罐体+顶）含义不同**。本算例统一用 Wh/Wc 表示几何宽度，避免符号冲突（v5 Sir 校对）。

### 7.3 单元力数值代入（P44）

**承压圈结构角度** α = 35°，sin α ≈ 0.574

```
T1 = (R/2) × [p + (W+F)/Ae] = 126.63 N/mm²         ...(B.4-7)
T2 = R·p − T1    = 135.77 N/mm²                    ...(B.4-8)
T3 = R·pa         = 164 N/mm²                      ...(B.4-9)
```

> **Q 的数值代入式**（5.5.2.3-2）：
>
> ```
> Q = T1·Wh + T2·Wc − T3·Ar·sin α = −8.024×10⁵ N   ...(B.4-10)
> ```
>
> ⚠️ **符号注意**：OCR 把负号漏掉，标准原文明确"Q < 0"（见 7.4 节），故 Q 实为 **−8.024×10⁵ N**。负号即"承压圈需承受压缩力"之意。

### 7.4 所需有效面积

**因 Q < 0（承压圈承受压缩力）**，按 SH/T 3167 式（5.5.2.3-2）计算 Ae：

```
Ae (承压圈所需) = |Q| / (Sa · φ) = 8.024×10⁵ / (137 × 0.75) = 7.81×10³ mm²  ...(B.4-11)
```

> **注**：本算例给出 Ae = 7.76×10³ mm²（基于 φ=0.85 的旧版结果），按 Sir 校对 φ=0.75 重算为 **Ae = 7.81×10³ mm²**（差异约 0.6%）。

### 7.5 实际有效面积（P45 第一次校核）

```
At (承压板对顶板有效面积) = (Wh + 150) × (tq − c) = 2.755×10³ mm²    ...(B.4-12)
Ac (承压板对罐壁有效面积) = Wc × (td − c)          = 798.519 mm²     ...(B.4-13)
A_t (有效总面积) = At + Ac                          = 3.554×10³ mm²  ...(B.4-14)
```

> **第一次校核**：A_t = 3.554×10³ mm² < Ae = 7.81×10³ mm² ✗ **不满足**！
> → 触发**迭代**：加大承压板厚度至 20 mm，重新计算 Wh, Wc, T1, T2, T3, Q, Ae

### 7.6 迭代收敛（P45 末）

加大承压板至 **20 mm** 后，重新计算：

| 项目 | 数值 | 单位 | 备注 |
|------|------|------|------|
| Wh | 207.35 | mm | 罐顶参与宽（迭代终值） |
| Wc | 242.914 | mm | 罐壁参与宽（迭代终值） |
| Q | −7.63×10⁵ | N | 总圆周力（迭代收敛） |
| At | 7.433×10³ | mm² | 承压板对顶板有效面积 |
| Ac | 9.147×10³ | mm² | 承压板对罐壁有效面积 |
| Ar | 4.86×10⁴ | mm² | 拱顶切线投影面积（OCR 中 "Ar" 实际为 R，参见 7.2 v6 修正） |
| **A_t** | **1.401×10⁵** | **mm²** | **总有效面积（迭代后）** |

> ⚠️ **迭代说明**：
> 1. **初始**：tq = 8 mm, td = 6 mm → Wh = 194.38 mm, Wc = 133.086 mm
> 2. **第一次校核**：A_t = 3.554×10³ mm² < Ae = 7.81×10³ mm² → 不合格
> 3. **加大承压板**：tq/tc 增大 → Wh, Wc 增大 → T1, T2, T3 重算 → Q 重算
> 4. **第二次校核**：A_t = 1.401×10⁵ mm² > Ae = 7.81×10³ mm² → 富余 17.9 倍 ✓
> 5. **承压板厚度 20 mm** 是承压圈最终结构（图 B.4-2）

**结论**：
- 抗风圈/承压圈取板厚 **20 mm**，按图 B.4-2 承压圈的最终结构示意施工
- **承压面积校核**：A_t = 1.401×10⁵ mm² > Ae = 7.81×10³ mm²，安全富余 **17.9 倍** ✓

### 7.7 真空稳定性 Von Mises 校核（v6 新增 — 6.5m 罐关键发现）

**⚠️ 重要发现**（2026-06-04 6.5m 己烷罐校核）：本算例（B.3）按压力算底圈仅需 **t_calc = 2.55~4.06 mm**，但 **真空稳定性 Von Mises 公式**要求 t ≥ 7.85 mm → 实际取 **8 mm**。**真空稳定性是低压储罐（小罐径 + 大长径比）壁厚的实际控制因素**。

**公式**（短圆筒 Von Mises，**API 620 5.10.5 不适用圆筒侧壁**）：

```
P_cr = 2.6 × E × (t/D)³ × [(D/L)² + 2(L/D)] / (1−ν²)
```

**安全系数要求**：SF = P_cr / P_vac **≥ 3.0**（GB 50341）

**算例对比**（6.5m 己烷罐，H=10m）：

| t 选用 (mm) | t_net (mm) | P_cr (kPa) | SF (vs 0.5 kPa) | 是否满足 |
|----|----|----|----|----|
| 6 | 4 | 0.48 | 0.96 | ✗ |
| 7 | 5 | 0.83 | 1.66 | ✗ |
| **8** | **6** | **1.62** | **3.24** | **✓** |
| 10 | 8 | 3.83 | 7.66 | ✓ |

**结论**：
- **6mm 板（净 4mm）**：SF=0.96 不满足，**禁止**
- **7mm 板（净 5mm）**：SF=1.66 不满足，**禁止**
- **8mm 板（净 6mm）**：SF=3.24 满足，**推荐**
- **10mm 板（净 8mm）**：SF=7.66 大幅富余，**保守方案**

**3 个常见陷阱**：
1. **不要用 Bresse 公式**（无限长圆筒）—— 算出来 SF=1.1，**实际 Von Mises 算 SF=4.1**（修正因子 3~5×）
2. **不要忽略 CA**——净厚 = 选用厚 - CA，否则 t³ 误差大
3. **罐内真空**与**外压**不同——GB 50341 真空工况需 SF≥3，API 650 注 a 仅要求 SF≥1.5

## 8. SH/T 3167 B.4 ↔ API 620 5.12.4 公式对照

> **核心价值**：本算例可作为 SH/T 3167 与 API 620 双向交叉核验的基准案例。

| 概念 | SH/T 3167 B.4 | API 620 5.12.4 | 备注 |
|------|---------------|----------------|------|
| 罐顶径向单位力 | T2 (B.2-2) | T_e = −P·R_roof/2 | 符号约定不同（SH/T 取正拉，API 取负拉） |
| 罐壁轴向单位力 | T1 (B.2-1) | T_os = −P·R | — |
| 罐顶环向单位力 | T2 (B.2-2) | T_φ = −P·R_roof/2 | 球面 T_e = T_φ |
| 侧壁环向单位力 | T1 (B.4-3) | T_os = −P·R | — |
| 罐顶有效板宽 | Wh (B.4-1) | w_t (Eq.24) | 公式结构完全相同 |
| 罐壁有效板宽 | Wc (B.4-2) | w_s (Eq.25) | 公式结构完全相同 |
| 顶板有效面积 | At = (Wh + 150) × t_roof | 由 w_t × t_roof 推得 | — |
| 罐壁有效面积 | Ac = Wc × t_shell | 由 w_s × t_shell 推得 | — |
| 总圆周力 | Q (B.4-6/B.4-10) | Q (Eq.26) | 结构相同，符号约定相反 |
| 所需有效面积 | Ae (B.4-11) | A_c (Eq.27) | 表达略有差异：SH/T 用 mm²，API 用 in² |
| 实际总面积 | A_t = At + Ac (B.4-14) | A_provided | — |

**符号约定差异（重要）**：
- **SH/T 3167**：压为负、拉为正（工程常规）
- **API 620 5.10.2.5**：拉为负、压为正（与工程常规相反）
- 代入公式时务必区分，避免符号错位

**符号命名差异（v5 新增）**：
- **SH/T 3167 公式 B.4-1/B.4-2**：用 Wh（罐顶参与宽）、Wc（罐壁参与宽）
- **API 620 Eq.24/Eq.25**：用 w_t（top）、w_s（shell）
- **SH/T 3167 公式 B.4-12/B.4-13**：用 At（顶板有效面积）、Ac（罐壁有效面积）
- **API 620**：用 A_roof_eff、A_shell_eff
- 本算例统一用 Wh/Wc/At/Ac 表示，避免与标准 5.3.2 的 W1/W2 重量项混淆

## 9. 关键提醒与坑

### 9.1 OCR 识别本算例的注意事项

1. **公式编号连续性**：B.2-1 → B.2-2 → B.2-3 → B.2-4 → B.3-1 → B.4-1 → ... → B.4-14，注意跨节编号
2. **几何 vs 重量的 W 符号区分**（v5 Sir 校对）：
   - **Wh**（本算例）：罐顶参与宽（几何尺寸，mm）
   - **Wc**（本算例）：罐壁参与宽（几何尺寸，mm）
   - **W1**（标准 5.3.2）：罐内液体重量（N）—— 本算例**不直接用**，避免混淆
   - **W2**（标准 5.3.2）：罐体和罐顶重量（N）—— 本算例**不直接用**，避免混淆
3. **下标易混**：
   - `T1`（轴向/承压圈）vs `T2`（环向/p·R）vs `T3`（R·pa）
   - `Ae`（B.2 中罐体内部横截面积 = π·R²）vs `Ae`（B.4-11 中承压圈所需有效面积）—— **不同阶段同名同符号**
   - `At`（顶板有效面积）vs `Ac`（罐壁有效面积）vs `A_t`（总有效面积）—— 大小写敏感
4. **希腊字母**：α（拱顶切线角）、ρ（密度）、φ（焊缝系数，**取 0.75**）、tq（顶板）、td（顶圈罐壁）
5. **压力符号**：p（总压力）、pa（设计压力）、pa1（1-1 平面附加压力）、p'（p+pa）
6. **A_e 数值单位陷阱**（v4 关键发现）：
   - OCR 显示 `Ae = π(R² − 2.112×10⁵)`，误读为 **Ae ≈ 2.11×10⁵ mm²**
   - 实际应为 **Ae = π·R² ≈ 2.11×10⁸ mm²**（中间 2.11×10⁵ 是小修正项）
   - 用 2.11×10⁸ 代入 B.2-1 才得到 T1 = 70.37 ✓
7. **W 数值陷阱**（v4 关键发现）：
   - OCR 显示 `W ≈ 1.792×10⁶ N` 或 `W = 1.472×10⁶ N`（罐体自重）
   - Sir 校对实际为 **W = 2.11×10⁷ N**（液体+罐体总重）
   - 用 2.11×10⁷ 代入 B.2-1 才得到 T1 = 70.37 ✓
8. **Wh/Wc 映射**（v4 新增，v5 重申）：
   - **Wh 对应 At**（罐顶一侧有效面积）
   - **Wc 对应 Ac**（罐壁一侧有效面积）
   - Wh 用 R_roof 算，Wc 用 R_shell 算
   - At = (Wh+150)·t_top, Ac = Wc·t_top_shell

### 9.2 数值校核要点

- **T1 = 70.37 N/mm² << T2 = 960.376 N/mm²**：T1 是叠加 (W+F)/Ae 与 p 的纵向净力；T2 是 p·R 的纯周向薄膜力
- **B.4-10 中 Q = −8.024×10⁵ N**（P44 初值，OCR 漏掉负号）→ 迭代终值 **Q = −7.63×10⁵ N**（P45）
  - 负号即承压圈需承受压缩力（compression ring）—— **负号 = 需承压圈，正号 = 需抗拉圈**
  - **v6 修正**：B.4-6 公式末项是 `T3·R·sin α`（R 罐壁中线半径），**不是 `Ar·sin α`**（OCR 误读）
- **A_t = 1.401×10⁵ mm² > Ae = 7.81×10³ mm²**（按 φ=0.75 重算）：富余 17.9 倍，安全裕度充足
- **迭代收敛**：P43→P44→P45 实质是 Wh, Wc 与 Q 的相互迭代校核，直到承压板厚度不再变化
- **焊缝系数 φ = 0.75**（Sir 2026-06-04 校对）：原 OCR 推测 0.85，按标准实际取 0.75 重算
- **真空稳定性 SF ≥ 3.0 是关键控制点**（v6 新增）：6.5m 己烷罐算例，压力仅需 2.55mm，真空稳定需 8mm，**真空稳定主导壁厚**
- **承压圈尺寸效应**（v6 新增）：Q ∝ R²，6.5m 罐（Q ≈ 0.12 MN）vs 16.4m 罐（Q ≈ 0.73 MN），**6.5m 罐 8mm 顶板即满足，无需独立承压板**

### 9.3 适用条件

- 本算例**仅适用于设计压力 pa ≤ 0.1 MPa 的低压储罐**（SH/T 3167 + SY/T 0608 双标准）
- 高压储罐（P > 0.1 MPa）须用 **GB 150** 体系
- 常压储罐（P ≤ 18 kPa）须用 **GB 50341 + SHT 3046** 体系

## 10. 工程延伸：与 API 620 90 kPa 算例的对比

| 项 | 本算例 (SH/T 3167, p=0.02 MPa) | API 620 90 kPa 算例 |
|----|--------------------------------|----------------------|
| 设计压力 | 0.02 MPa | 0.09 MPa |
| 罐径 | 16.4 m | 6.5 m |
| 介质 | 乙二醇 (ρ=1100) | 己烷 (RVP 高) |
| 材料 | S30408 (Sa=137) | Q345R (Sa=159) |
| 承压圈板厚 | 20 mm | 16 mm（圈板 240mm 高）|
| 压缩环面积 | 圈板几何 A_t=1.4×10⁵ mm² | 详算 A_c=3800 mm² |
| 富余度 | 17.9× (A_t/Ae) | 4~5× |

> **结论**：低压档（0.02 MPa）下承压圈板厚由构造（20 mm）控制，而非强度；90 kPa 接近 API 620 上限时，承压圈板厚由强度控制，须按 5.12.4 详算。

### 10.1 承压圈尺寸效应（v6 新增 — 6.5m 罐校核发现）

**核心规律**：`Q ∝ R²`（承压圈总圆周力与罐壁中线半径平方成正比）

| 罐径 D (m) | 罐壁中线 R (m) | Q (kN) | Ae (mm²) | 需独立承压板？ | 6mm 顶板 A_t (mm²) | 富余倍数 |
|------|------|------|------|------|------|------|
| 6.5 | 3.255 | **−120** | 1 010 | ❌ 不需要 | 2 059 | 2.03× ✓ |
| 10.0 | 5.005 | −285 | 2 390 | ❌ 8mm 顶板可代偿 | ~2 760 | 1.15× ⚠ |
| 16.4 | 8.200 | **−802** | 7 810 | ✅ 需 20mm 独立承压板 | 3 554 | 0.45× ✗ |

**应用判据**：
```
if (A_t / Ae) ≥ 2: 6mm 顶板代偿, 无需独立承压板
elif (A_t / Ae) ≥ 1: 8mm 顶板勉强, 需校核
else: 需独立承压板 (10~20mm)
```

**Sir 2026-06-04 实战结论**：6.5m 罐径 + 0.02 MPa 压力下，**罐顶-罐壁自然连接（8mm 顶板 + 8mm 罐壁顶圈）已满足承压圈要求 2.95×**。无需 16.4m 算例那样的 20mm 独立承压板。
