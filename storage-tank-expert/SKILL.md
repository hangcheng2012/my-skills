## 立式圆筒形储罐 vs 过程设备（核心判据：设备型式）

**拆分原则**：**不按设计压力判断，按设备型式判断**。**立式圆筒形储罐**（无论常压、低压、高压）归本 skill；**过程设备**（反应器、换热器、塔器、卧式容器、球罐等）归 `pressure-vessel-expert`。

### A. 设备型式（按《压力容器设计工程师培训教程》附录 A 表 A.2）

按罐顶结构分为三大类：

| 大类 | 罐顶型式 | 支撑方式 | 细分型式 |
|------|---------|---------|---------|
| **外浮顶储罐** | 浮顶（上下浮动）| — | 单盘式 / 双盘式 |

# 核心工作模式

触发关键词：储罐、API650、API620、拱顶罐、浮顶罐、常压罐、油罐、罐区

## 1. 常压储罐模式（GB 50341 / SH/T 3046 / API 650）

**适用**：常压储罐（≤18 kPa）

**核心设计因素**：液柱静压（分圈设计）、风载荷、地震载荷、真空/呼吸工况（API 2000）、沉降与基础

**关键设计点**：
- 壳板逐层厚度递减设计（按 1.5m / 1.8m 分圈）
- 罐底设计（边缘板 + 中幅板，腐蚀裕量 2mm）
- 罐顶与罐壁连接结构（包边角钢/抗拉环/通气罩）
- 环向应力为主控制

## 2. 低压储罐模式（API 620 / SH/T 3167 / SY/T 0608）

**适用**：18 kPa < P ≤ 0.1 MPa

**核心设计因素**：内压、罐壁分圈、**承压圈（压缩环）** ← 自动调用子 skill

**关键设计点**：拱顶推力、锚栓 6 工况校核、边缘板宽、Von Mises 真空稳定（SF≥3）

# 储罐专项设计规则

## 锚栓/地脚螺栓设计

- **数量最低**：8 个（4 的倍数）
- **直径最低**：M24
- **材质**：Q235B（构造）或 Q345B
- **埋深**：≥ 600mm，双螺母防松

**锚栓座局部应力分析**：Troitsky 模型 / ASME VIII-2 线性化 / CBFEM / Pushover

详见 `references/internal-anchor-reinforcement-ring.md`

## 罐壁分圈设计

按 1.5m 或 1.8m 分圈，每圈独立计算厚度，取该圈最大压力：

```
t = (P * R) / (S * E - 0.6 * P) + CA
```

**制造最小厚度**：碳钢 6mm，不锈钢 5mm

## 罐底设计

- **底板组成**：边缘板（环形）+ 中幅板（中心）
- **厚度**：max(6mm, 强度计算) + CA(2mm)
- **边缘板宽度**：600~1000mm

## 罐顶与罐壁连接

**包边角钢**：L75×75×6 ~ L100×100×8（按罐径选取）

**罐底开孔/排水槽**：补强板角焊缝 100% 真空盒检漏

详见 `references/api-620-bottom-sump-drainage.md`

# 工程风险提示

- 壁厚接近最小规范值 / 地震区 / 外压/真空工况
- 焊缝不可达区域 / 腐蚀裕量不足 / 基础沉降
- 罐底边缘板腐蚀 / 罐顶角钢强度不足 / 拱顶失稳 / 浮顶通气不足


# 承压圈（压缩环）自动调用

储罐设计涉及承压圈计算时，自动调用 `sht-3167-shell-compression-ring`：
- 罐壁分圈厚度 + 承压圈校核（型 a/b 角钢、型 f 罐壁加厚+顶板延伸）
- 仅做承压环计算，不含锚栓/抗风/罐底（由本 skill 处理）
