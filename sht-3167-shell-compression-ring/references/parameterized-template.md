# SH/T 3167 罐壁+承压圈 参数化计算模板

> 本文件提供**完整可复制**的参数化计算模板，包含 5 圈 / 7 圈分圈、厚板/薄板、Q345R/S30408 等常见组合。

## 1. 标准输入参数表

```python
# 罐体几何 (mm)
D = ?           # 内直径
H = ?           # 罐壁高度
H_a = ?         # 最高液位
R_roof = 0.85*D # 拱顶半径 (推荐 0.85D)

# 介质
rho = ?         # 密度 (kg/m³)
g = 9.81        # 重力加速度

# 设计条件
p_a = ?         # 设计压力 (MPa), 0.018~0.1
T = ?           # 设计温度 (℃)
p_test = 1.25 * p_a  # 试验压力

# 材料
material = ?    # 'Q345R' / 'S30408' / 'Q245R' / '16MnR'
S_a = ?         # 许用应力 (MPa), 查 SH/T 3167 + SY/T 0608 附表
E = 2.06e5      # 弹性模量
nu = 0.3

# 焊接 / 腐蚀
phi = 0.75      # 焊缝系数 (SH/T 3167 默认, Sir 校对)
CA = 2          # 腐蚀裕量 (碳钢 2 / 不锈钢 0)

# 分圈
n_courses = 5   # 5 圈 (1.5~2.0m 分圈)
# 或 n_courses = 7 (短罐)
```
